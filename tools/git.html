# 常用命令

  • 查看分支git branch -a: 全部分支 -r: 远程分支

  • 拉取远端分支到本地git fetch origin <远程分支名x>:<本地分支名x>

  • 切换分支: git checkout <name>

  • 创建分支并切换至新分支git checkout -b <name>

  • 查看文件状态：git status

  • add 所有修改 git add .

  • 提交本地分支 git push

  • 合并某分支到当前分支git merge <name>

  • 删除分支 git branch -d <name>

  • 丢弃工作区修改git checkout -- <file>

  • 撤销add，返回工作区git reset HEAD <file>

  • 提交 git commit -m "message"

  • 查看提交记录日志 git log --pretty=online

  • 打标签：git tag <tag> <headID>

  • 查看某次提交的详细内容 git show <headID>

  • 撤销commit git 中HEAD表示当前版本，上一个版本就是 HEAD^。前100个版本就是HEAD~100 git rest --hard HEAD^

  • 首次提交

    git remote add origin https://github.com/xx
git push -u origin master
