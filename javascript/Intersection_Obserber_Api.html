# Intersection Obserber API

异步观察目标元素的可视状态或者两个元素的相对可视状态

使用场景

  • 当页面滚动时，懒加载图片或其他内容
  • 实现“可无限滚动”网站，也就是当用户滚动网页时直接加载更多内容，无需翻页
  • 根据用户是否已滚动到相应区域来灵活开始执行任务或动画

交集检测通常需要涉及到事件监听，以及对每个目标元素执行Element.getBoundingClientRect() 方法以获取所需信息。
