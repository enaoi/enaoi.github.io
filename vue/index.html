# vue生命周期

这些 生命周期的 hooks 可以 emit 它们自己的 自定义事件。前缀是 hook:

监听子组件的hooks,这个方法对于第三方的插件非常实用～～

<template>
    <Child @hook:mounted="childMounted"></Child>
</template>
